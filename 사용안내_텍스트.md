[26년도 신입 인턴 턴 교환 프로그램]

인턴 선생님들의 원활한 턴 교환을 위해
교환 프로그램을 제작하였습니다.

- 초기 비밀번호: 1234 (로그인 후 변경 바랍니다.)

- 기본 교환 규칙
  1) 필수과목 5개 유지 (IM, GS, OB, PE, 진로탐색)
     : 휴가턴을 제외하고 각 1턴 이상 필요
  2) 본원(분당) 비율 50% 이상
     : 전체 13턴 중 분당 근무 최소 7턴 유지
  3) 1턴·2턴 교환 불가
     : 3턴~13턴만 교환 가능
  4) 휴가턴 교환 시 수지 밸런스
     : 상대에게 휴가턴을 줬으면,
       같은 상대에게서 다른 휴가턴을 받아야 함

- 모든 규칙은 시스템이 자동 검증합니다.
  규칙 위반 교환은 신청 자체가 불가능하니
  안심하고 사용하세요!


🔐 1. 로그인

• 이름: 본인 이름 입력 (예: 홍길동)
• 비밀번호: 기본 비밀번호 1234
• 첫 로그인 시 비밀번호 변경 화면이
  자동으로 나타납니다. 꼭 변경해주세요!
• 이후에도 사이드바 > 🔑 비밀번호 변경
  에서 언제든 변경 가능

💡 화면 전환
   오른쪽 상단 📱/💻 버튼으로
   모바일 ↔ PC 레이아웃 전환 가능


📱 2. 화면 구성

로그인 후 화면은 크게 두 영역입니다.

  메인 화면 (가운데)
  → 📤 교환 신청 / 📋 전체 스케줄 / 🏪 교환 장터

  사이드바 (왼쪽 >> 버튼)
  → 👤 내 현황 / 📩 받은·보낸 요청 / 🧪 교환 시뮬레이션


🔄 3. 교환 신청하기

[STEP 1] 상대방 & 턴 선택
  • 교환할 상대방 이름 선택
  • 교환할 턴 선택 (3턴~13턴)
  • "나: ○○ ↔ 상대: ○○" 미리보기 확인

[STEP 2] 조건 확인
  • ✅ 초록색 = 교환 가능!
  • ❌ 빨간색 = 규칙 위반

[STEP 3] 요청 보내기
  • 📨 요청 보내기 버튼 클릭
  • 상대방에게 교환 요청이 전달됩니다

⛔ 1턴·2턴은 교환 불가 턴입니다!


📋 4. 전체 스케줄 보기

모든 인턴의 턴표를 한눈에 볼 수 있어요.
• 노란 배경 = 나의 행
• 파란 배경 = 교환 신청한 상대방
• 빨간 테두리 = 교환 중인 턴
• 연보라 배경 = 휴가 턴

🔄 다른 사람의 교환 내용을 보려면
   새로고침 버튼을 눌러주세요


🏪 5. 교환 장터

직접 상대를 찾기 어려울 때
게시판에 교환 요청을 올려보세요!

▸ 장터 둘러보기
  • 🟢 초록 점 = 나와 바로 교환 가능
  • 🔴 빨간 점 = 조건 불충족으로 교환 불가
  • 가능한 조합이 있으면 응하기 버튼 표시

▸ 내 턴 올리기 (두 가지 방식)
  ① 줄 턴 지정
     "내가 ○턴을 줄게, ○○ 받고 싶어"
  ② 받을 턴 지정
     "○○ 받고 싶어, 내 아무 턴이나 줄게"

▸ 내 게시물 관리
  • 취소 버튼으로 게시물 삭제 가능
  • 교환 완료 시 자동으로 마감 처리


📩 6. 요청 수락 / 거절 (사이드바)

사이드바를 열면 받은 요청을 확인할 수 있어요.
• ✅ 수락 → 교환 즉시 실행 (구글 시트 자동 반영!)
• ❌ 거절 → 요청 거절

💡 새로운 요청이 오면
   사이드바에 숫자로 알림이 표시돼요


🧪 7. 교환 시뮬레이션 (사이드바 하단)

신청 전에 어떤 교환이 가능한지
미리 탐색할 수 있어요.

  🔄 특정 턴 교환
  → 특정 턴 기준 교환 가능한 파트너 전체 목록

  🎯 특정 턴 받기
  → 원하는 과목을 받을 수 있는 모든 조합 탐색

• 요청 버튼 → 바로 교환 신청으로 연결
• ⏳ 표시 = 이미 대기 중인 요청 있음


⚠️ 꼭 알아두세요!

• 필수과목: IM, GS, OB, PE, 진로탐색
  → 교환 후에도 5개 과목 모두 유지 (휴가턴 제외 각 1턴 이상)
• 교환 불가 턴: 1턴, 2턴
• 분당 근무: 최소 7턴 이상 유지
• 휴가턴 교환: 줬으면 같은 상대에게서 다른 휴가턴 받아야 함
• 실시간 반영: 교환 완료 시 구글 시트 자동 반영
• 새로고침: 다른 사람 교환 확인 → 🔄 버튼 클릭
• 비밀번호: 기본 1234 → 꼭 변경해주세요!

궁금한 점이 있으면 언제든 문의해주세요!
